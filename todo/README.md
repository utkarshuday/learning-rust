# Todo List

This is a fast and simple command-line task manager built with Rust. It uses the Clap crate for a smooth terminal experience and saves your tasks directly to a local file for easy persistence. Designed for speed, it lets you quickly add, toggle, or clear your list without any unnecessary bloat.
